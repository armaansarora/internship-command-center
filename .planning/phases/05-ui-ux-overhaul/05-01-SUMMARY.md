## Decisions Made
- **useRef strict mode fix:** React 19 requires explicit initial value for useRef -- changed `useRef<T>()` to `useRef<T | undefined>(undefined)` in FrozenRouter's usePreviousValue hook
- **Extended toast coverage:** Added toast to status-editor.tsx and notes-editor.tsx (detail page components) beyond the plan's explicit list, as they also call server actions and should show feedback
- **Toast deduplication:** Used explicit `id` parameter on status and notes editor toasts to prevent flooding when users make rapid changes
- **Explicit dark theme:** Passed `theme="dark"` to Toaster component since the app uses `forcedTheme="dark"` in ThemeProvider

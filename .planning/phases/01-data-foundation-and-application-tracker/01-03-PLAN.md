         - Zod schema: `{ id: z.number(), notes: z.string() }`
         - Parse, validate, update: `db.update(applications).set({ notes, updatedAt: new Date() }).where(eq(applications.id, id)).run()`
         - Revalidate both paths
         - Return `{ success: true }` or `{ error: string }`

       - `createApplication(formData: FormData)`:
         - Zod schema: `{ company: z.string().min(1), role: z.string().min(1), tier: z.enum(['T1','T2','T3','T4']).optional(), sector: z.string().optional(), platform: z.string().optional(), notes: z.string().optional() }`

         - If sector not provided, infer from tier: T1 => 'RE Finance', T2 => 'Real Estate', T3 => 'Finance', T4 => 'Other'

         - Revalidate `/applications`

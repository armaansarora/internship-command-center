Create `src/lib/calendar.ts`:

1. `listUpcomingEvents(days = 7)`:
   - Call `getGoogleClient()` to get calendar
   - `timeMin` = now ISO string, `timeMax` = now + days in ISO string
   - Call `calendar.events.list({ calendarId: "primary", timeMin, timeMax, maxResults: 10, singleEvents: true, orderBy: "startTime" })`
   - Map results to `CalendarEvent` type: `{ id, summary, description, start (ISO string), end (ISO string), location, htmlLink }`
   - Handle all-day events: use `event.start?.date` if `event.start?.dateTime` is undefined
   - Wrap in try/catch, return empty array on failure

2. `createInterviewEvent(opts: { company: string, role: string, startDateTime: string, endDateTime: string, location?: string, description?: string })`:
   - Call `getGoogleClient()` to get calendar
   - Create event with:
     - summary: `"Interview: ${company} - ${role}"`
     - description: `opts.description || "Interview for ${role} position at ${company}"`
     - location: `opts.location`
     - start: `{ dateTime: opts.startDateTime, timeZone: "America/New_York" }`
     - end: `{ dateTime: opts.endDateTime, timeZone: "America/New_York" }`
     - reminders: `{ useDefault: false, overrides: [{ method: "popup", minutes: 30 }, { method: "email", minutes: 60 }] }`
   - Call `calendar.events.insert({ calendarId: "primary", requestBody: event })`
   - Return `{ success: true, eventId: res.data.id, htmlLink: res.data.htmlLink }` or `{ success: false, error }`

3. `createFollowUpReminder(opts: { company: string, role: string, dueDate: string, note?: string })`:
   - Call `getGoogleClient()` to get calendar
   - Create ALL-DAY event with:
     - summary: `"Follow up: ${company} - ${role}"`
     - description: `opts.note || "Follow up on ${role} application at ${company}"`
     - start: `{ date: opts.dueDate }` (YYYY-MM-DD format, no time = all-day)
     - end: `{ date: opts.dueDate }`
     - reminders: `{ useDefault: false, overrides: [{ method: "popup", minutes: 480 }] }` (8 hours = morning reminder)
   - Return `{ success: true, eventId, htmlLink }` or `{ success: false, error }`

Export type: `CalendarEvent`

Create `src/lib/calendar-actions.ts`:
- `'use server'` directive

1. `addInterviewToCalendar(formData: FormData)`:
   - Extract: company, role, date (YYYY-MM-DD), time (HH:MM), duration (minutes, default 60), location (optional)
   - Validate with zod: company and role required, date and time required
   - Construct startDateTime and endDateTime ISO strings in America/New_York timezone
   - Call `createInterviewEvent()`
   - Return `{ success, htmlLink }` or `{ error }`

2. `addFollowUpToCalendar(company: string, role: string, dueDate: string, note?: string)`:
   - Call `createFollowUpReminder({ company, role, dueDate, note })`
   - Return `{ success, htmlLink }` or `{ error }`

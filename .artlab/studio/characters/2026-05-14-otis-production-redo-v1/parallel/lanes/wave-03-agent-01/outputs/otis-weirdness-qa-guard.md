# Otis Weirdness QA Guard

Lane: wave-03-agent-01
Artifact type: ship-ready QA guard for prompt-only production probe

## Pre-Generation Guard

- Generate one identity reference first, not a seven-pose sheet.
- Prefer transparent source output or clean alpha-ready neutral background over chroma-key.
- Request native high-resolution source with 4096 px minimum long edge when the tool allows it.
- Keep the call-button token in the prompt as "tiny," "matte brass," "worn," "secondary," and "attached to a short vest watch chain."
- Include the negative prompt guard every time the token appears.

## Source Preflight Guard

Block expansion if any probe fails:

- Long edge below 4096 px, unless the parent coordinator explicitly revises the source contract.
- Missing alpha or unusable alpha/chroma separation.
- Cropped hands, feet, chain, token, ledger, or keycard.
- Token size larger than palm or visually louder than face.
- Chain or token overlaps fingers in a way that cannot be cleanly masked.
- Face, beard, belly, or posture drift away from the approved soft Santa concierge identity.

## App-Scale Guard

Check generated candidates at these display sizes before expanding:

- 64 CSS px: Otis should read as warm older concierge; token may disappear.
- 96 CSS px: silhouette, belly, beard, and greeting posture should read.
- 144 CSS px: token can read as a small brass speck or chain detail, not as a pendant.
- 192 CSS px and up: token should be legible as a worn call-button cap without adding visual noise.

## Motion Compatibility Guard

The token is compatible with CharacterStage only if:

- Idle: token rests still, no required animation.
- Listening: optional tiny hand-to-chain pose works without covering face or belly.
- Thinking: one subtle thumb rub works as a static pose and as a low-amplitude loop.
- Talking: token stays still while hand gesture carries expression.
- Alert: token recedes; alert read comes from posture and eyes.
- Working: token does not overlap ledger/keycard anatomy.

## Promotion Blocker Reminder

This lane creates prompt and QA artifacts only. No generated image source, preflight JSON, staged derivative, review board proof, or public/art promotion is authorized from this lane.

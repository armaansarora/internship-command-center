# The Tower — Mascot Concept Prompts (GPT Image 2, one-shot hardened)

*Six distinct mascot "visions." **Six separate prompts — one per character**, each fully self-contained
for a **fresh one-shot session** (no follow-up edits assumed). Save renders to
`docs/research/mascots/` as `1-tower.png … 6-seal.png` (variants `-a`, `-b`). I'll judge them with you,
revise the losing prompts, and build the winner into `/lobby-pilot`.*

**Hardening (the hyper-critical pre-mortem).** Each prompt now defends against GPT Image 2's usual
failure modes: it slaps on a busy *environment* (→ "isolated, flat backdrop, no scenery/floor"); drifts
*cartoonish/toy/emoji* (→ "refined, not a toy/kids-cartoon/emoji/sticker"); leaks *off-palette colors or
chrome* (→ "ONLY these 3 hexes, no chrome"); over-detailing that *dies at small size* (→ "clean bold
silhouette that still reads as a tiny app icon"); *renders text* (→ "no text/letters/numbers/logo");
*ignores negations* (→ positive framing, esp. the Key); *uncanny faces* (→ "appealing, not uncanny").
Sources: [OpenAI guide](https://developers.openai.com/cookbook/examples/multimodal/image-gen-models-prompting-guide) ·
[fal.ai](https://fal.ai/learn/tools/prompting-gpt-image-2).

## 1 · The Tower, alive  *(the building given a soul)*
```
Brand mascot for a premium app — clean logo-style packshot, 1:1.

Subject: a friendly anthropomorphic Art-Deco skyscraper with ELEGANT TALL proportions (graceful, not a
squat blob) — a tapering gold tower of three stepped setbacks rising to a flat keystone cornice. Two
large, calm, gently glowing oval eyes set high on the facade (warm and kind, never spooky); a tall warm
cream-lit arched doorway at the base reading as a soft closed-mouth smile; a few rows of tiny warm
window-lights. A slight forward lean, as if attentively greeting you. ONE single clean tower — no city,
no other buildings.

Render: stylized 3D, Pixar/Blender feel — smooth premium surfaces, soft bevels, subtle subsurface
scattering; refined and luxurious. NOT a toy, kids-cartoon, emoji, or sticker.
Colors: ONLY deep navy #1A1A2E, brushed gold #C9A84C, warm cream #F5F1E8 — no other hues, no chrome, no neon.
Light: soft cinematic three-point studio light, warm key upper-left, cool rim, gentle ambient occlusion;
warm light glows from the doorway and windows.
Frame: one isolated subject, centered, full body, slight 3/4 angle, seamless flat deep-navy #1A1A2E
backdrop — no scenery, floor, or pedestal. Clean bold silhouette that still reads shrunk to a tiny app icon.
No text, letters, numbers, logos, or watermark.
```

## 6 · The Seal  *(premium emblem — the "starbucks" slot)*

# ⏳ Verdicts — the open ledger of pending ground truth

*Every Verdict-Pending campaign (doctrine §3) records a falsifiable outcome prediction here
at Debrief. Every Briefing checks this file for predictions now resolvable. Lessons from
these campaigns graduate to doctrine ONLY when the verdict resolves — never from Siege-time
self-assessment. A class kit with no resolved verdicts behind its rubric stays marked
**unvalidated**. Cap: 200 lines.*

Format, one entry per prediction:

```
## <date> · <campaign codename>
- Prediction (falsifiable): <what will be true in the world if the artifact was right>
- Check-back: <date or triggering event>
- Resolution: (pending) | <date>: <what actually happened> → AAR reopened: <link>, lesson graduated: <yes/no/what>
```
